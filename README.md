# a11y-paradise

**Comprehensive accessibility training and tools — from 25 years of real-world experience.**

## About This Project

a11y-paradise is an open-source collection of accessibility knowledge, training materials, and developer tools built from over 25 years of hands-on experience in digital accessibility — including work as a web developer and head of accessibility at the [Canadian National Institute for the Blind (CNIB)](https://www.cnib.ca/).

This project exists to ensure that hard-won, practical accessibility expertise is freely available to:

- **Developers** who want to build accessible products from the start
- **Accessibility professionals** looking for tested approaches and reference materials
- **Organizations** seeking to embed accessibility into their culture and processes

## What's Here

### [Training](training/)

Structured learning materials across four areas:

| Area | Description |
|------|-------------|
| [Standards](training/standards/) | Deep dives into WCAG, ARIA, PDF/UA, EN 301 549, and other accessibility standards |
| [Developer Guides](training/developer-guides/) | Practical how-to guides for building accessible web applications, components, and patterns |
| [Testing](training/testing/) | Manual and automated testing methodology, tools, and workflows |
| [Organizational Practices](training/organizational/) | Building accessibility culture, procurement policies, and "born accessible" practices |

### [Tools](tools/)

Open-source developer tools for accessibility testing, auditing, and remediation.

### [Resources](resources/)

Curated references, reading lists, and links to the broader accessibility community.

# Training

This directory contains all training content for a11y-paradise, organized into four areas that cover the full scope of digital accessibility practice.

| Area | What You'll Find |
|------|-----------------|
| [Standards](standards/) | WCAG, ARIA, PDF/UA, EN 301 549 — understanding what the standards require and why |
| [Developer Guides](developer-guides/) | Practical techniques for building accessible web applications and components |
| [Testing](testing/) | How to test for accessibility — manual techniques, automated tools, and workflows |
| [Organizational Practices](organizational/) | Embedding accessibility into organizational culture, procurement, and process |
